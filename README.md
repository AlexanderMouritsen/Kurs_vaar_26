# Kurs_v-r_26

## Forutsetninger

Før du starter kurset må du ha følgende installert på datamaskinen din.
Hvis du ikke allerede har lastet ned disse, finner du lenkene nedenfor:

- **Node.js** – JavaScript-kjøretidsmiljø som trengs for å kjøre serverside-kode.
  👉 Last ned her: [https://nodejs.org/](https://nodejs.org/)

- **SQLiteStudio** – Grafisk verktøy for å administrere SQLite-databaser.
  👉 Last ned her: [https://sqlitestudio.pl/](https://sqlitestudio.pl/)

> **Merk:** Sørg for at begge programmene er riktig installert før du fortsetter med kurset.
